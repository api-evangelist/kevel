# Kevel (kevel)
Kevel is a set of APIs that allows customers to build a unified ad platform supporting any ad format, any creative, and management of multiple demand sources. It provides Decision API, Management API, and Reporting API for building custom ad serving infrastructure.

## APIs

### Kevel Decision API
The Decision API enables ad requests without using ad code. By posting to a RESTful endpoint, Kevel's ad engine returns decision data and creative contents for serving ads in your application.

**Human URL:** [https://dev.kevel.com/reference/request](https://dev.kevel.com/reference/request)
